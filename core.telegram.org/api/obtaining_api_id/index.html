Creating your Telegram Application

我们欢迎所有的开发人员免费使用我们平台的API 和源代码来开发类似Telegram的即时消息App。

为了确保Telegram生态系统的一致性和安全性，
所有第三方客户端App都必须遵守API服务条款。

获取 api_id

获取 API id 步骤如下：

  • Sign up for Telegram using any application.
  • Log in to your Telegram core: https://my.telegram.org.
  • Go to ‘API development tools’ and fill out the form.
  • You will get basic addresses as well as the api_id and api_hash parameters required for user authorization.
  • For the moment each number can only have one api_id connected to it.

我们会给你的电话号码发送重要的开发者通知，所以请用最新的手机号码激活你的Telegram账号。

使用api_id

请注意：所有客户端类库均收到严格监控，以防止滥用。

如果您使用Telegram接口用来刷屏，滥发邮件，伪造订阅者和频道用户数量，会被永远封禁。

基于对过度滥用Telegram接口的考虑，所有非Telegram官方客户端账号 都会 被监视，以避免违反 服务条款。

如果意外被禁，可以邮件找recover@telegram.org 解释您的使用API的意图，要求解禁。
请注意，电子邮件是人工审查，因此自动生成的电子邮件会被过滤。

使用Telegram开源代码

欢迎使用我们的开源代码。代码里有个示例API ID。此 API ID在服务器端收到限制，不适合发布给终端用户，会导致API_ID_PUBLISHED_FLOOD错误。发布App前，必须要获得自己的 API id。

请记住也要发布您的代码，以遵守GNU GPL许可。
